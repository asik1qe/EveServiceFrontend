# EVE Online Frontend

Frontend part of the EVE Online project.  
This application provides a convenient interface for viewing, filtering, and analyzing LP Store offers, market-related data, and other information collected and processed by the backend service.

---

## About the project

This repository contains the client-side application for the EVE Online project.  
The frontend is responsible for displaying processed data received from the backend API, providing sorting, filtering, and a user-friendly way to explore offers and item information.

The goal of the project is to make complex in-game economic and LP Store data easier to browse and compare through a clean web interface.

---

## Features

- View LP Store offers by corporation
- Sort offers by different fields
- Filter and search through data
- Responsive interface for desktop and mobile devices
- Convenient card-based presentation of offers
- Integration with backend API
- Error handling for unavailable data
- Flexible UI for future expansion
